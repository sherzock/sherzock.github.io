Bobobo's bizarre adventure

  • Website

Bobobo's bizarre adventure is a 2D plataformer game made for a university subject. The game was done by a group of 2 students using C++ with SDL.

This project was great to have a first introduction to several concepts of engine programming as I learned how to create UMLs and how to create different modules in order for the engine to work properly and be more accessible and maleable later on. I also got to learn about gameplay programming and how certain thigns work, for example simple pathfindings for entities to move using A*, Dijkstra and BFS, or how we could handle certain mechanics or elements easier when I was coding.
